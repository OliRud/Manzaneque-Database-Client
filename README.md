# Manzaneque Database Client
This is a graphical event-driven client designed to provide a front for helpdesk operatives within a real-estate agency.

Developed using C# with a MySQL backend database, I've built this program to demonstrate database connectivity, manipulation and complex data selection to provide the features a helpdesk operative would need. I've also developed a prototype for security levels to implement role authorisation for certain features.

This progaram allows you to:

~ Create and resolve tickets

~ View the contents of the 6 main tables

~ Execute custom queries

~ Compile monthly reports to summarise the cotntents of the database

~ Backup the 6 main tables as CSV files

This program is incomplete and many features are still missing.
